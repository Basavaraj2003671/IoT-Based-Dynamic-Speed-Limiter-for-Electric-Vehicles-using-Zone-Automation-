# IoT-Based-Dynamic-Speed-Limiter-for-Electric-Vehicles-using-Zone-Automation-
This project implements an automatic EV speed-limiting system based on zone automation using nRF24L01 RF modules and Arduino. A transmitter placed in restricted zones (ex: School Zone) sends a signal to the EVâ€™s receiver unit. Upon detection, the EV automatically reduces its speed to a safe limit, alerts the driver, and prevents speeding inside the zone. When the vehicle exits the zone, normal speed control is restored.

Features

Automatic speed limiting in restricted zones

Smooth, gradual speed reduction

Buzzer + LED safety alerts

Real-time LCD monitoring

Manual speed buttons with restricted control inside zones

Fail-safe timeout handling

Tech Used

Arduino, NRF24L01, LCD I2C, L298N, DC Motor, Buzzer, IoT & RF communication.

Use Cases

Smart Mobility, EV Safety Systems, Smart Traffic Zones, Intelligent Transportation.
